# CodeProjectsHub

## Overview
This repository contains modular, classroom-ready projects across Python, C, and Unix Shell. Each project is self-contained with documentation, test suites, and structured source code for easy onboarding and reproducibility.

## Project Highlights

### CProjects
- **01_Huffman_File_Compressor**: Huffman-based file compression tool with Unity test framework.

### PythonProjects
- **01_LTE_5G_Log_Analysis**: Parses and analyzes LTE/5G logs.
- **02_NR_5G_Param_Calculator**: Computes NR 5G parameters using modular Python.
- **03_NR_5G_Throughput_Calculator**: Calculates theoretical throughput for NR 5G.
- **04_Laptop_Monitor**: Monitors system metrics via CLI.
- **05_PDF_to_Audio_File_Converter**: Converts PDF content to MP3 using TTS.
- **06_AI_Resume_Intelligence_Extractor**: Extracts structured data from resumes using AI-powered logic.

### UnixShellProjects
- **01_Clean_List_Automation**: Cleans and formats messy lists.
- **02_Disk_Usage_Warning**: Alerts on disk usage thresholds.
- **03_Executable_File_Locator**: Custom `which` command implementation.
- **04_Fixing_File_Dir_Names**: Renames files and directories with spaces.

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/sanjeet9019/CodeProjectsHub.git
	```
2. Navigate to any project folder:
	```bash
   cd PythonProjects/05_PDF_to_Audio_File_Converter
	```	
2. Follow instructions in its README.md or docs/specs.md.
